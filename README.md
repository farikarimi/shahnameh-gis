# Shahnameh GIS

A deep map visualizing geospatial information extracted from the epic poem Shahnameh (The Book of Kings) written by the Persian poet Ferdowsi between c. 977 and 1010 CE. This project is being developed by Farimah Karimi within the framework of the seminar "Modelling and Digital Maps" (Modellierung und digitale Karten) taught by Professor Ã˜yvind Eide (Institute for Digital Humanities) at the University of Cologne.
